Hack phpbb3 integration basic V Beta 0.9.0 - 2009-07-30
--------------------
Requirements:
xbtit 2.0.557 (without HACKs is better)
phpBB 3.0.5 (without MODs is better)
xbtit and phpBB3 installed in the same MySQL database 
phpBB 'Account activation:' disabled
phpBB 'Allow username changes:' disabled
phpBB 'Allow e-mail address re-use:' disabled

ADDED - phpBB3 database prefix option
ADDED - phpBB3 path
ADDED - the admin have to set the path of phpBB3,useful for some feature as to delete phpbb3 cache
ADDED - registration in phpBB3 for any new account of xbtit
ADDED - update last members and total users on phpBB3
ADDED - export xbtit's members into phpBB3 forum
ADDED - import phpBB3's members into xbtit tracker
ADDED - the admin can enable/disable the 'Export/Import members function' in  'Tracker's Settings'
ADDED - the admin can disable 'Account activation/Allow username changes/Allow e-mail address re-use' on phpbb3 in 'Tracker's Settings' and he can try to delete the phpbb3 cache relative to 'Account activation/Allow username changes/Allow e-mail address re-use'. (have to set the forum path correctly)
ADDED - when the user changes the password in his xbtit's profile,the system update the password on phpbb3, too
ADDED - when the user recovers the password with xbtit,the system updates both the passwords,on xbit and phpbb3,too
ADDED - when the user changes the email,timezone,language in xbtit's profile,the system update also the profile in phpBB3
ADDED - the admin can try to hide 'Edit account settings' on phpbb3 in 'Tracker's Settings' and also to delete the cache (only for prosilver and subsilver2 style and for english language)
ADDED - the admin can try to restore the orginal 'Edit account settings' on phpbb3 user's profile in 'Tracker's Settings' and also to delete the cache. Have to set the forum path correctly (only for prosilver and subsilver2 style and for english language)
ADDED - check last version on "Admin Panel"
ADDED - forum showed into tracker's page
CHANGED - nothing
DELETED - nothing
FIXED - nothing
OPTIM - nothing
