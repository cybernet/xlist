Arc90_Service_Twitter_Response

Description
Description | Vars (details) | Methods (details)

Arc90_Service_Twitter_Response represents a response to a Twitter API call.

In addition to the $_data property (accessible via $response->data or $response->getData()), an Arc90_Service_Twitter_Response provides the following read-only attributes related to the original HTTP request:

  • url
  • content_type
  • http_code
  • header_size
  • request_size
  • filetime
  • ssl_verify_result
  • redirect_count
  • total_time
  • namelookup_time
  • connect_time
  • pretransfer_time
  • size_upload
  • size_download
  • speed_download
  • speed_upload
  • download_content_length
  • upload_content_length
  • starttransfer_time
  • redirect_time
