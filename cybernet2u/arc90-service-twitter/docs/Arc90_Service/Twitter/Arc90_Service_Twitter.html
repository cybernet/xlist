Arc90_Service_Twitter

Description
Description | Vars (details) | Methods (details) Constants (details)

Arc90_Service_Twitter provides methods for interacting with the Twitter API.

Based on Twitter's online documentation.

NOTE: At the time of this writing, clients are allowed 100 requests per 60 sixty minute time period, starting from their first request. For more information, please see the API documentation regarding rate limiting.
